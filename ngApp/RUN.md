## Available Scripts

In the project directory, you can run:

### `npm install`

Create production build
### `ng build`

Run production server
### `npm run prod`

Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

Development mode
### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
